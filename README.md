# kirubakaran2's CTF Writeups Site

Welcome to my CTF writeups!  
Visit the live site: https://kirubakaran2.github.io

## Features

- Classic retro terminal theme
- Gamified "unlock" style challenge list
- Writeups and challenge walkthroughs (WIP)

## To add a new writeup:

1. Edit `script.js` and add your new challenge writeup in the `writeups` object.
2. Add or unlock the challenge in `index.html`.
3. Push changes. Your site updates automatically!
